Please follow the steps below:

1) Open the command promt (type cmd in the search bar)

2) Navigate to the folder src inside Report

3) Compile the files :
javac *.java

4) Simulate the results :
C:\Users\thatava2\workspace\Report\src> java HOFComputation bd_nit_pareto_CR.txt 100 2

The first argument is the path to the file
The second argument is the number of computations to be used
The third argument is whether to use the decay factor or not - 1 for yes, any other number for not using it
